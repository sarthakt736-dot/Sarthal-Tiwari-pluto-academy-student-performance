# 🎓 Student Performance Analysis
**Pluto Academy | Data Analytics Internship — Project 02**

**Developed by:** Sarthak Tiwari

## Problem Statement
EDA on 1,000 student exam records to identify factors affecting grades,
segment at-risk students, and recommend school interventions.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## Dataset
Students Performance in Exams — Kaggle (1,000 records)

## Key Findings
- 17.5% students are at-risk (score < 50 in any subject)
- Test prep completion boosts scores by ~6 points average
- Master's degree parents → 10.1 pts higher scores than some high school
- Reading ↔ Writing correlation: r = 0.955 (very strong)
- Free/reduced lunch students have 2× higher at-risk rate
